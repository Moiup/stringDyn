# stringDyn_concat_new(\_,\_)

Concats two strings by returning a new one.

## **Declaration**

```C
char *stringDyn_concat_new(
                        char *s1,
                        char *s2
                     )
```

## **Parameters**
**s1**  
The first string.

**s2**
The second string.

## **Result**
Returns the concatenation of `s1` with `s2`. `NULL` if failed.
