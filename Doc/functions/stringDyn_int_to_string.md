# stringDyn_int_to_string(\_)

Converts an int into a string.

## **Declaration**

```C
char *stringDyn_int_to_string(
                        int n
                    )
```

## **Parameter**
**n**  
The integer to rewrite as a string.


## **Result**
Returns the string representing the integer `n`. Returns `NULL` if failed.
